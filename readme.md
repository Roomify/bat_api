# Booking and Availability Management API

Module port of Bat API for Drupal 8


### URL Operations

- Delete (DELETE)
  bat_api/availability_event/{availability_event}
  bat_api/availability_state/{availability_state}
  bat_api/property/{property}
  bat_api/unit/{unit}
  bat_api/unit_type/{unit_type}

- Retrieve (GET)
  bat_api/availability_event/{availability_event}
  bat_api/availability_state/{availability_state}
  bat_api/property/{property}
  bat_api/unit/{unit}
  bat_api/unit_type/{unit_type}

- Index (GET)
  bat_api/availability_event
  bat_api/availability_state
  bat_api/property
  bat_api/unit
  bat_api/unit_type

- Create (POST)
  bat_api/availability_event
  bat_api/availability_state
  bat_api/property
  bat_api/unit
  bat_api/unit_type

- Update (PATCH)
  bat_api/availability_event/{availability_event}
  bat_api/availability_state/{availability_state}
  bat_api/property/{property}
  bat_api/unit/{unit}
  bat_api/unit_type/{unit_type}

- View (GET)
  bat_api/availability_event/{availability_event}/view
  bat_api/availability_state/{availability_state}/view
  bat_api/property/{property}/view
  bat_api/unit/{unit}/view
  bat_api/unit_type/{unit_type}/view
